# 项目名称

> 项目简介

## 快速开始

```bash
# 安装依赖
npm install

# 开发模式
npm run dev

# 构建生产版本
npm run build
```

## 技术栈

- **游戏框架**：Phaser 3 (v3.80.1)
- **构建工具**：Vite (v5.4.11)
- **开发语言**：JavaScript (ES6+)

## 项目结构

```
project/
├── src/               # 源代码
│   ├── main.js        # 游戏入口
│   ├── config.js      # 配置常量
│   ├── entities/      # 游戏实体
│   ├── scenes/        # 场景
│   └── systems/       # 系统模块
└── README.md          # 项目说明
```
