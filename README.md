# 御剑无双 (Sword Immortal)

> 东方仙侠 × 简约几何 × 无双割草 × Roguelike

一款在浏览器中直接游玩的无双割草游戏。以简约的几何美术风格承载东方仙侠的意境，结合 Roguelike 随机强化系统，带来独特的战斗体验。

## 游戏特色

- **独特视觉**：简约几何与仙侠元素的创新融合
- **爽快战斗**：大量敌人 + 华丽技能 + 连击系统
- **高可重玩**：Roguelike 随机强化，每局不同体验
- **即开即玩**：浏览器直接运行，无需下载
