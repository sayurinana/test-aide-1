# 第二章：世界观与背景设定

## 2.1 世界观框架

### 天地设定
**九州仙界**——一个仙道昌盛的世界。

千年前，天地灵气充沛，修仙者众多。然而，一场名为"妖潮"的灾难打破了平静。妖魔从幽冥界涌出，侵蚀九州大地。各大仙门联手镇压，最终将妖潮封印于"镇妖塔"之下。

如今，封印渐弱，妖潮再度涌动。作为新一代剑修，你将踏入镇妖塔，以剑为媒，以身镇妖。

### 核心设定
- **灵气**：天地间的能量，修仙之本
- **剑道**：以剑为器，驾驭灵气的修炼体系
- **妖魔**：被负面情绪和怨气滋养的邪恶生物
- **镇妖塔**：封印妖潮的神秘建筑，每层都有更强的妖魔

## 2.2 主角设定

### 身份背景
**无名剑修**——镇妖塔的守护者

- 出身：普通山村，有剑道天赋
- 师承：镇妖塔前任守护者
- 使命：守护封印，斩尽妖魔
- 特点：沉默寡言，以剑代言

- **普通状态**：淡然白色
- **战斗状态**：边缘发出淡青色光芒
- **受伤状态**：闪烁红色警示
- **释放技能**：发出强烈的青/金色光芒

## 2.3 敌人来源设定

### 妖魔分类

| 类型 | 来源 | 视觉特征 | 代表 |
|------|------|----------|------|
| **怨灵** | 人类怨念 | 漂浮的三角形，暗紫色 | 飘影、怨魂 |
| **妖兽** | 被妖气侵蚀的动物 | 四边形身体，尖锐棱角 | 妖狼、蛇妖 |
| **魔将** | 强大的妖魔首领 | 复杂几何组合，大型 | 各层 Boss |
| **邪修** | 堕落的修仙者 | 类人几何形，暗红色 | 精英敌人 |

### 妖魔视觉设计原则
1. **几何抽象**：用简单几何形状表现
2. **颜色区分**：
   - 小怪：暗紫、灰色
   - 精英：暗红、深蓝
   - Boss：黑金、血红
3. **动态感**：通过旋转、缩放、粒子表现生命力
4. **威胁感**：尖锐的角度、锋利的边缘

## 2.4 视觉融合方案

### 简约几何 + 东方仙侠

**设计理念**：以极简的几何形状承载东方仙侠的意境

| 元素 | 几何表现 | 仙侠寓意 |
|------|----------|----------|
| 剑 | 细长矩形 + 发光效果 | 剑气、锋芒 |
| 人物 | 三角 + 菱形组合 | 道袍、道冠 |
| 技能 | 圆形、弧线、光束 | 真气、剑气 |
| 敌人 | 尖锐多边形 | 妖魔的邪恶 |
| 场景 | 渐变背景 + 云纹线条 | 仙境、妖域 |
