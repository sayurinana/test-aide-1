# 第一章：游戏概念

## 1.1 游戏名称
**《御剑无双》**（暂定）
- 英文名：Sword Immortal

## 1.2 游戏类型
- 主类型：无双割草 (Musou / Hack and Slash)
- 子类型：Roguelike + 无尽生存

## 1.3 核心概念
> "一剑破万敌，每战皆新生"

在东方仙侠世界中，玩家扮演一名剑修，面对源源不断的妖魔大军。每次战斗都是一次修炼，通过获取随机强化不断提升实力，挑战更高的波次记录。

## 1.4 核心循环 (Core Loop)

```
┌─────────────────────────────────────────────────────────┐
│                                                         │
│   战斗 ──→ 击杀敌人 ──→ 波次结束 ──→ 选择强化 ──┐      │
│     ↑                                           │      │
│     └───────────────────────────────────────────┘      │
│                                                         │
│   直到：角色死亡 → 结算 → 新一局游戏                    │
│                                                         │
└─────────────────────────────────────────────────────────┘
```

**单局流程**：
1. 开始新游戏
2. 进入战斗场景
3. 击杀当前波次所有敌人
4. 波次结束，选择 1/3 强化
5. 进入下一波次（难度递增）
6. 重复 3-5 直到死亡
7. 结算界面（显示统计、分数）
8. 返回主界面或立即重开

## 1.5 目标玩家
- **核心玩家**：喜欢爽快战斗、追求高分记录的玩家
- **休闲玩家**：想要短时间体验割草快感的玩家
- **Roguelike 爱好者**：喜欢随机构筑、策略选择的玩家

## 1.6 游戏时长
- **单局时长**：5-15 分钟（取决于玩家水平）
- **会话时长**：15-30 分钟（适合碎片时间）

## 1.7 游戏节奏
| 波次 | 节奏 | 敌人数量 | 特点 |
|------|------|----------|------|
| 1-5 | 热身 | 10-30 | 熟悉操作，建立节奏 |
| 6-10 | 上升 | 30-50 | 难度渐增，开始构筑 Build |
| 11-20 | 高潮 | 50-100 | 考验操作和 Build 质量 |
| 21+ | 极限 | 100+ | 极限挑战，追求记录 |

## 1.8 差异化特点
1. **简约几何 + 仙侠**：独特的视觉风格融合
2. **短平快体验**：单局时间短，爽感密度高
3. **Roguelike 深度**：每局都有不同的 Build 可能
4. **浏览器即玩**：无需下载，打开即玩
