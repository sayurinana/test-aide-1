## 核心内容

### gdd-index.md - GDD 总索引

**项目基本信息**：
- 游戏名称：御剑无双 (Sword Immortal)
- 游戏类型：无双割草 + Roguelike + 无尽生存
- 平台：Web 浏览器
- 美术风格：简约几何 + 东方仙侠

**核心循环**：
```
战斗 → 击杀敌人 → 波次结束 → 选择强化 → 下一波次 → 循环
                                        ↓
                              角色死亡 → 结算
```

**技术栈**：Phaser 3 + Vite + JavaScript

**系统摘要**：
- 角色：6 基础属性 + 5 隐藏属性 + 5 技能
- 敌人：4 小怪 + 1 精英 + 1 Boss
- Roguelike：4 级稀有度 + 20+ 强化道具
- 无尽模式：4 种波次类型 + 里程碑奖励

### gdd-chapter1.md - 游戏概念

**核心概念**：「一剑破万敌，每战皆新生」

**单局流程**：
1. 开始新游戏 → 进入战斗场景
2. 击杀当前波次所有敌人
3. 波次结束，选择 1/3 强化
4. 进入下一波次（难度递增）
5. 重复直到死亡 → 结算

**目标玩家**：
- 核心：喜欢爽快战斗、追求高分
- 休闲：短时间体验割草快感
- Roguelike 爱好者：随机构筑、策略选择

**游戏时长**：
- 单局：5-15 分钟
- 会话：15-30 分钟

### gdd-chapter2.md - 世界观与背景设定

**世界观**：九州仙界
- 千年前妖潮灾难，被封印于镇妖塔
- 如今封印渐弱，玩家作为剑修守护封印

**主角设定**：无名剑修
- 简约几何风造型：菱形头部 + 三角形道袍
- 配色：主体白/浅灰 + 剑为青色

**视觉融合方案**：
| 元素 | 几何表现 | 仙侠寓意 |
|------|----------|----------|
| 剑 | 细长矩形 + 发光 | 剑气、锋芒 |
| 人物 | 三角 + 菱形 | 道袍、道冠 |
| 技能 | 圆形、弧线 | 真气、剑气 |
| 敌人 | 尖锐多边形 | 妖魔邪恶 |

**配色方案**：
- 主角：#E8E8E8 浅灰白 + #4ECDC4 青色剑光
- 敌人：#6B5B95 暗紫(小怪) / #C0392B 暗红(精英)
- 特效：#FF6B6B 红(伤害) / #FFE66D 黄(暴击)

### gdd-chapter3.md - 核心玩法设计

**操作方式**：
- 移动：WASD / 方向键
- 普攻：左键 / J
- 技能：Q/E/R/Space
- 闪避：Shift

**连击系统**：
| 连击数 | 奖励 |
|--------|------|
| 10+ | 伤害 +5% |
| 50+ | 伤害 +15%，移速 +10% |
| 100+ | 伤害 +25%，攻击带溅射 |
| 200+ | 伤害 +40%，进入无双状态 |

**技能设计**：
1. **剑斩**（普攻）：100% ATK，前方扇形 60°
2. **剑气横扫**（Q）：150% ATK，半径 150px 圆形，4s CD
3. **瞬步斩**（E）：200% ATK，直线 200px，5s CD，位移无敌
4. **护体真气**（R）：格挡 1 次伤害，反弹 100% ATK，8s CD
5. **剑域**（Space）：50% ATK/0.5s，持续 3s，25s CD
6. **御风步**（Shift）：100px 闪避，0.3s 无敌，1.5s CD

**伤害公式**：
```
最终伤害 = 基础ATK × 技能系数 × (1+连击加成) × (1+强化加成) × 暴击倍率 × 随机波动(0.9~1.1)
```

### gdd-chapter4.md - 角色系统设计

**基础属性**：
| 属性 | 初始值 | 上限 |
|------|--------|------|
| HP | 100 | 无限 |
| ATK | 10 | 无限 |
| DEF | 5 | 80%减伤 |
| SPD | 200px/s | 500px/s |
| CRIT | 5% | 100% |
| CRITDMG | 150% | 500% |

**隐藏属性**：
- 攻击速度 (1.0x)
- 攻击范围 (1.0x)
- 冷却缩减 (0%, 上限75%)
- 生命恢复 (0/s)
- 吸血 (0%)

**角色状态机**：
普通 ↔ 战斗中 → 受伤 → 濒死
技能/闪避期间 → 无敌
连击 200+ → 无双

### gdd-chapter5.md - 敌人系统设计

**敌人类型**：

| 类型 | 代号 | HP | ATK | SPD | 行为特点 |
|------|------|-----|-----|-----|----------|
| 飘影 | E01 | 10 | 5 | 100 | 直线追踪 |
| 妖狼 | E02 | 15 | 8 | 180 | 蓄力冲锋 |
| 蛇妖 | E03 | 12 | 6 | 80 | 远程吐息 |
| 怨魂 | E04 | 20 | 4 | 120 | 死亡分裂 |
| 邪修 | E05 | 80 | 15 | 100 | 精英，3技能 |
| 妖将 | B01 | 500 | 20 | 60 | Boss，双阶段 |

**难度缩放**（波次 w）：
```
敌人HP = baseHP × (1 + w × 0.1)
敌人ATK = baseATK × (1 + w × 0.08)
敌人SPD = baseSPD × (1 + w × 0.02)
```

**波次配置**：
- 波次 1-3：飘影为主，教学
- 波次 5/10/15...：Boss 波
- 波次 10/20/30...：狂潮波（数量翻倍）

### gdd-chapter6.md - Roguelike 系统设计

**稀有度分级**：
| 稀有度 | 颜色 | 概率 |
|--------|------|------|
| 普通 | 白色 | 60% |
| 稀有 | 蓝色 | 25% |
| 史诗 | 紫色 | 12% |
| 传说 | 金色 | 3% |

**强化道具（共20+）**：

属性类（7个）：
- A01 锋锐：ATK +3
- A02 坚韧：HP +15
- A07 不动明王（传说）：ATK +20%, HP +30%

技能类（7个）：
- S03 横扫千军：剑气横扫范围 +50%
- S07 万剑归宗（传说）：新终极技能

特效类（7个）：
- E04 处刑者（史诗）：低HP敌人伤害 +50%
- E06 轮回（传说）：死亡复活一次

生存类（6个）：
- H04 吸血：造成伤害回复 3% HP
- H06 不死金身（传说）：HP<10%时受伤-50%

**Build 方向**：
- 暴击流：专注 + 猛击 + 处刑者
- 攻速流：迅捷 + 快速冷却 + 连斩
- 范围流：剑气延长 + 横扫千军
- 生存流：回春 + 吸血 + 金钟罩
- 技能流：剑道通神 + 快速冷却

### gdd-chapter7.md - 无尽模式框架设计

**波次类型**：
| 类型 | 频率 | 奖励 |
|------|------|------|
| 普通波 | 每波 | 三选一强化 |
| 精英波 | 每3波 | 额外掉落 |
| Boss波 | 每5波 | 稀有强化保底 |
| 狂潮波 | 每10波 | 双倍奖励 |

**难度阶段**：
| 阶段 | 波次 | 设计目标 |
|------|------|----------|
| 热身 | 1-5 | 熟悉操作 |
| 发展 | 6-10 | 构筑Build |
| 挑战 | 11-20 | 考验Build |
| 极限 | 21+ | 追求记录 |

**里程碑奖励**：
- 波次5：额外强化选择
- 波次10：传说保底
- 波次15：全属性+10%
- 波次25：HP全恢复+传说
- 波次50：排行榜殿堂

**评分公式**：
```
最终分数 = (击杀×10 + 波次×1000) × 连击系数 × 效率系数 × 难度系数
```

### gdd-chapter8.md - UI/UX 设计

**界面流程**：
加载 → 主界面 → 战斗 ↔ 暂停/强化选择 → 结算 → 主界面

**战斗HUD布局**：
- 顶部：HP条、波次、击杀数
- 底部：技能栏（Q/E/R/Space）+ 闪避
- 左下：连击数
- 战斗区：伤害数字浮动

**强化选择界面**：
- 三选一卡片布局
- 键盘 1/2/3 快速选择
- 稀有度颜色区分

**设置项**：
- 音频：主音量、音乐、音效
- 画面：粒子效果、屏幕震动、伤害数字
- 操作：按键提示显示

**响应式设计**：
- 基准：1920×1080
- 支持：1280×720、移动端
