# 子计划 2：技术原型开发

## 目标
搭建游戏技术框架，实现基础游戏循环，验证核心技术可行性。

## 具体步骤

### 2.1 项目初始化
- 创建项目结构
- 配置 Vite + Phaser 3 开发环境
- 配置 TypeScript（可选）
- 设置热重载开发流程

### 2.2 游戏场景搭建
- 实现游戏主场景
- 实现基础相机系统
- 实现场景边界

### 2.3 玩家角色实现
- 实现角色渲染（简约几何风格）
- 实现键盘/鼠标移动控制
- 实现基础动画状态机
- 实现角色朝向系统

### 2.4 基础攻击系统
- 实现普通攻击
- 实现攻击动画
- 实现攻击范围可视化（调试用）

### 2.5 敌人基础实现
- 实现敌人渲染
- 实现敌人生成器
- 实现基础 AI（追踪玩家）
- 实现敌人群体管理

### 2.6 碰撞检测系统
- 实现玩家与敌人碰撞
- 实现攻击与敌人碰撞
- 实现碰撞伤害处理

### 2.7 基础 UI
- 实现生命值显示
- 实现击杀计数
- 实现简单调试信息

## 验证标准
- [ ] 项目可正常运行
- [ ] 玩家可控制移动
- [ ] 玩家可进行攻击
- [ ] 敌人可生成并追踪玩家
- [ ] 攻击可消灭敌人
- [ ] 基础 UI 显示正常
- [ ] 性能满足 60 FPS（100+ 敌人）

## 依赖
- 前置：子计划 1（前期设计）
- 后续：子计划 3（核心系统实现）
