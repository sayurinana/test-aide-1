# 子计划 2：品牌重塑 + 技能系统

## 目标

将游戏从《御剑无双》更名为《无尽幸存者》，移除仙侠元素，简化技能系统为 4 个通用技能。

## 具体步骤

### 品牌重塑

1. 更新游戏名称：
   - `index.html`：标题改为「无尽幸存者 | Endless Survivor」
   - `MainMenuScene.js`：主菜单标题
   - `package.json`：项目名称
   - `README.md`：项目说明
   - `vite.config.js`：base 路径（如需要）

2. 移除仙侠文案：
   - 技能名称：剑气横扫 → 通用名称
   - 敌人名称：飘影、妖狼、蛇妖等 → 通用名称
   - UI 文案：所有「仙」「剑」「御」相关文字

3. 调整视觉元素：
   - 技能特效颜色：从仙侠风（青/金）调整为通用风格
   - 保留几何美术风格基础

### 技能系统简化

1. 设计 4 个通用技能：
   - **加速** (Speed Boost)：短时间移动速度大幅提升
   - **闪现** (Dash)：瞬间位移到指定方向
   - **无敌** (Shield)：短时间免疫所有伤害
   - **治疗** (Heal)：恢复一定比例生命值

2. 设计技能 SVG 图标：
   - 加速：风/箭头元素
   - 闪现：闪电/瞬移轨迹
   - 无敌：盾牌/护罩
   - 治疗：十字/心形

3. 实现技能图标：
   - 创建 `src/assets/skills/` 目录
   - 编写 4 个 SVG 文件
   - 集成到 `HUDScene.js` 技能栏

4. 重构 `SkillManager.js`：
   - 移除原有 5 技能
   - 实现新的 4 技能逻辑
   - 更新技能配置到 `config.js`

5. 更新强化系统：
   - 移除 `BuffData.js` 中与旧技能相关的强化
   - 添加新技能相关强化（冷却缩减、效果增强等）

## 验证标准

- [ ] 游戏所有界面显示新名称「无尽幸存者」
- [ ] 无任何仙侠相关文案残留
- [ ] 4 个技能正常工作（Q/E/R/Space）
- [ ] 技能图标在 HUD 正确显示
- [ ] 技能相关强化可正常获取和生效
